Alphabet

Here is a list of the basic 26 English language characters of the alphabet and three main extra sounds.

The alphabet are letters that we use to create words which then create or describe concepts and things.
They are basic building blocks of words and are represented by sounds and symbols.
The English language uses a character alphabet of symbols which have their own sounds. Other languages like Chinese use pictorial characters which are more complex than English characters.

Main 26

A a - Name = Aye/ai. Sound = ah.

B b - Name = Bee. Sound = bh.

C c - Name = See. Sound = ch/kh.

D d - Name = Dee. Sound = dh

E e - Name = Ee. Sound = eh.

F f - Name = Ef. Sound = fh.

G g - Name = Gee. Sound = gh.

H h - Name = Haych/heich. Sound = huh.

I i - Name = Eye. Sound = ii.

J j - Name = Jay/jae/jey. Sound = jh.

K k - Name = Kay/key/kae. Sound = kh(Same as c).

L l - Name = El. Sound = lh.

M m - Name = Em. Sound = mh.

N n - Name = En. Sound = nh.

O o - Name = Ou/Ow. Sound = oh.

P p - Name = Pee. Sound = ph.

Q q - Name = Que/cu/cue. Sound = kh/kw.

R r - Name = Ar. Sound = rh.

S s - Name = Es. Sound = ss.

T t - Name = Tee. Sound = th.

U u - Name = Yu. Sound = uh/ah.

V v - Name = Vee. Sound = vh.

W w - Name = Doubleyu. Sound = Wh.

Y y - Name = Why/wi. Sound = yh.

X x - Name = Ecs. Sound = ks/sh.

Z Z - Name = Zed/zee. Sound = zh(Similar to ts).

Extra sounds

Ch ch - Name = None. Sound = ch.

Sh sh - Name = None. Sound = sh.

Th th - Name = None. Sound = th.
